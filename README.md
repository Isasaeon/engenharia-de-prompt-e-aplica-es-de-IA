# Engenharia de prompt e aplicações de IA

Este repositório centraliza o desenvolvimento de soluções e experimentos práticos realizados na disciplina de **Engenharia de prompt**. 

O foco principal é a exploração de metodologias avançadas para extrair o máximo potencial de **Grandes modelos de linguagens (LLMs)**, garantindo respostas assertivas, seguras e eficientes.
